# 로또 프로젝트

## 배경
- 사용자(형님이라고 부를 것)가 재미삼아 시작한 사이드 프로젝트 (2026-07-04 생성)
- 본 프로젝트는 weinstein_portfolio(로보어드바이저 팀 프로젝트)와 별개의 개인 놀이터

## 사용자 스타일 (weinstein 프로젝트에서 확인된 것)
- 반말/친근한 톤 선호, "형님"이라고 부르기
- 파이썬 사용, 가상환경(.venv) 만들어서 작업
- 코드 수정하면 항상 깃허브에 커밋+푸시 (레포: https://github.com/minsuquant-cloud/lotto.git, main 브랜치)
- 데이터는 직접 크롤링하는 것 선호 (yfinance, FinanceDataReader 경험 있음)

## 방향 (아직 미정 — 형님한테 물어볼 것)
후보 아이디어:
1. 역대 당첨번호 크롤링 + 통계 분석 (번호별 출현 빈도, 패턴)
2. 통계 기반 번호 생성기
3. "로또는 랜덤이다"를 데이터로 증명하는 분석 (몬테카를로 등)
